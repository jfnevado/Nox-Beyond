## Description

Nox Beyond is a NFT Dao, they wanted the report in their investment in a DashBoard to have more insight and a overall view.

## Steps
- Data retrieval through APIs

  - CoinGecko
- Data cleaning
- Data transformation
- Data visualization

## Tools used
- Python
- PowerBI
